# 拡張三和音構造におけるアルゴリズム的生成とネオ・リーマン理論の応用：SUHMMLydianから九の和音における変形操作の包括的研究

## 1. 序論：無限の拡張と和声の幾何学

西洋音楽の和声理論は、歴史的に「垂直方向の拡張」のプロセスとして記述することができる。中世の二声によるオルガヌムから、バロック・古典派における三和音（トライアド）の確立、そしてロマン派からジャズ・現代音楽に至る四和音（セブンスコード）およびテンション・ノート（ナインス、イレブンス、サーティーンス）の飽和へと、和音の構成密度は増大の一途をたどってきた。現代の音楽理論、特にジェイコブ・コリアー（Jacob Collier）のような現代の理論家や、数理音楽理論（ネオ・リーマン理論）の領域では、伝統的な機能和声のラベル付け（トニック、ドミナントなど）だけでは説明しきれない構造的論理が出現している。

本報告書は、提示された以下の二つの主要な問いに対する包括的かつ徹底的な分析を提供するものである。第一に、長三度と短三度を交互に無限に積み重ねるアルゴリズム的生成プロセスと、それに付随する「Super Ultra Hyper Mega Meta Lydian」等の概念的枠組みについて。第二に、九の和音（ナインスコード）において特定の構成音（奇数番目または偶数番目）を保持しつつ、他の構成音を半音移動させる変形操作（トランスフォーメーション）の理論的定義についてである。

これらの現象は、単なる「コード進行」や「スケール」の問題にとどまらず、音楽を幾何学的な空間として捉える**変形理論（Transformational Theory）**や、生成規則に基づく**アルゴリズム的和声（Algorithmic Harmony）**の核心に触れるものである。本稿では、これらの概念を歴史的背景、音響物理学的根拠、そして現代ジャズ理論やネオ・リーマン理論の観点から多角的に解剖し、その全貌を明らかにする。

## 2. アルゴリズム的和声生成：交互三度積層と「Super Ultra」複合体

最初の問いは、ルートから長三度（Major 3rd, M3）と短三度（Minor 3rd, m3）を交互に積み重ねる操作に関するものである。これは、静的な「和音」の定義を超え、再帰的な規則を適用して無限の和声内容を生成するアルゴリズムの領域に属する。

### 2.1 SUHMMLydianの構築（M3・m3交互積層）

ルートから長三度、短三度、長三度、短三度……と交互に積み重ねて構成されるコード（例：C, E, G, B, D, F#, A）は、現代のインターネット音楽理論コミュニティやジャズ理論の最前線において、**Super Ultra Hyper Mega Meta Lydian**（スーパー・ウルトラ・ハイパー・メガ・メタ・リディアン、しばしば**SUHMMLydian**と略される）として広く認識されている。この名称と概念は、グラミー賞受賞アーティストであり多楽器奏者の**ジェイコブ・コリアー（Jacob Collier）**によって一般化されたものである1。

#### 2.1.1 構造的導出と五度圏の同型性

SUHMMLydianの決定的な特徴は、それがダイアトニック・スケール（7音）やクロマティック・スケール（12音）のような「閉じたシステム」ではなく、理論上無限に拡張可能な「螺旋（スパイラル）」構造を持つ点にある。その構築ルールは以下の通りである：

$$Note_{n+1} = Note_n + Interval_n$$

ここで、$Interval_n$は4半音（長三度）と3半音（短三度）を交互に繰り返す。ルートをC（ハ）とした場合の生成プロセスを以下に詳述する。

|**ステップ**|**生成される音**|**ルートからの音程**|**音程幅（半音）**|**五度圏上の位置（C起点）**|
|---|---|---|---|---|
|基点|**C**|Root|0|0|
|1 (M3)|**E**|Major 3rd|4|時計回り4ステップ|
|2 (m3)|**G**|Perfect 5th|7 (4+3)|時計回り1ステップ|
|3 (M3)|**B**|Major 7th|11 (7+4)|時計回り5ステップ|
|4 (m3)|**D**|Major 9th|14 (11+3)|時計回り2ステップ|
|5 (M3)|**F#**|\#11th|18 (14+4)|時計回り6ステップ|
|6 (m3)|**A**|Major 13th|21 (18+3)|時計回り3ステップ|
|7 (M3)|**C#**|\#15th (b2)|25 (21+4)|時計回り7ステップ|

この生成プロセスから得られる最初の7音（C, E, G, B, D, F#, A）は、**C Major 13(#11)**コードを形成する。この構成音は、**リディアン・モード（Lydian Mode）**の音階（1, 2, 3, #4, 5, 6, 7）と完全に一致するため、名称に「リディアン」が含まれる1。

ここで最も重要な理論的洞察は、**「長三度と短三度を積み重ねることは、完全五度（Perfect 5th）を積み重ねることと音響的・構造的に同義である」**という点である3。

数学的には、$4 \text{ semitones} + 3 \text{ semitones} = 7 \text{ semitones}$（完全五度）となる。したがって、この交互三度積層（Alternating Thirds Stack）は、**五度圏（Circle of Fifths）**の順序を並べ替えたものと同型（Isomorphic）であるといえる。

通常の五度圏（C-G-D-A-E-B-F#...）と比較すると、交互三度積層は五度圏を「スキップしながら」進む螺旋階段のようなものである。CからE（五度圏で4つ先）、EからG（五度圏で3つ戻る...ではなく、和音内では累積的に機能するが、生成原理としては五度圏のシャープ側（時計回り）の音を順次回収していくシステムとして機能する）6。

#### 2.1.2 「無限」の命名法と概念的意義

「Super Ultra Hyper Mega Meta」という極端な修飾語は、このスケールの再帰性と無限性を表現したジョーク混じりの記述である1。通常のリディアン・モードにおいて、「明るさ（Brightness）」の拡張は#4（CリディアンにおけるF#）で停止する。しかし、五度圏のパターン（あるいは交互三度のパターン）をそのまま継続すれば、次はC#（#1あるいは#15）、その次はG#（#5）といった具合に、次々とシャープが付与されていく。

- **Lydian:** \#4まで（F#）
    
- **Super Lydian:** 次のシャープ（C#）を追加
    
- **Ultra Lydian:** さらに次のシャープ（G#）を追加
    
- **Hyper/Mega/Meta:** このプロセスを無限に継続し、全12音、さらには微分音的な領域（純正律におけるピッチのズレなど）へ拡張していく概念
    

したがって、この操作は**「五度圏に基づく無限生成プロセスを、三度堆積のヴォイシングで具現化したもの」**と定義できる9。

### 2.2 その対極：SUHMMDorianの構築（m3・M3交互積層）

次に、ユーザーが提示したもう一つの和音、すなわちルートから**短三度（m3）**と**長三度（M3）**を交互に積み重ねる操作（例：C, Eb, G, Bb, D, F, A）について考察する。

#### 2.2.1 構造分析とドリアン・モード

この順序（m3 $\to$ M3）での生成プロセスを追跡する：

1. **C** (Root)
    
2. C + m3 = **Eb** (Minor 3rd)
    
3. Eb + M3 = **G** (Perfect 5th)
    
4. G + m3 = **Bb** (Minor 7th)
    
5. Bb + M3 = **D** (Major 9th)
    
6. D + m3 = **F** (Perfect 11th)
    
7. F + M3 = **A** (Major 13th)
    

生成されるセット（C, Eb, G, Bb, D, F, A）は、**Cm13**コードを形成する。音階として見れば、これは**ドリアン・モード（Dorian Mode）**（1, 2, b3, 4, 5, 6, b7）と完全に一致する。

M3始動のスタックが五度圏の**時計回り（シャープ方向・明るい方向）**に進むのに対し、m3始動のスタックは五度圏の**反時計回り（フラット方向・暗い方向）**に進む性質を持つ。

- C + m3 = Eb（五度圏で反時計回りに3ステップ）
    
- このプロセスを続けると、Ab, Db, Gb...とフラットが増加していく。
    

#### 2.2.2 理論的名称：「Super Ultra Hyper Mega Meta Dorian」

コリアーの「SUHMMLydian」に対応する形で、このm3始動のスタックは理論コミュニティにおいて**Super Ultra Hyper Mega Meta Dorian**（あるいは**SUHMMDorian**）と呼ばれている11。

スニペット12における定義によれば、「任意のモードのSuper Ultra...は、そのモードの第1音から第4音または第5音への繰り返しパターンから生じる」とされる。m3から始まるスタックは、自然なマイナー響き（ドリアン）を生成し、継続することで五度圏のフラット側へ無限に螺旋を描く。これは、SUHMMLydianの**ネガティブ・ハーモニー（Negative Harmony）**的鏡像であり、明るさの極致へ向かうリディアンに対し、暗さの深淵へ向かうドリアンという対比構造を持つ13。

### 2.3 操作自体の名称と分類

「SUHMMLydian」は固有名詞的な呼び名であるが、この「操作」自体を指す音楽理論用語としては、以下のものが適切である。

1. Chain of Thirds（三度の連鎖） / Circle of Thirds（三度圏）
    
    これが最も一般的かつ形式的な名称である5。ただし、通常の「長三度圏」は3音（C-E-G#-C）で閉じ、「短三度圏」は4音（C-Eb-Gb-A-C）で閉じてしまう（ディミニッシュコード等）。本件のような**「交互三度（Alternating Thirds）」による連鎖は、オクターブ内で閉じずに無限に続くため、「無限三度連鎖（Infinite Chain of Thirds）」や「生成的三度螺旋（Generative Spiral of Thirds）」**と呼ぶのが正確である。
    
2. シリンガー・システムにおける「第4グループ」
    
    理論家ジョセフ・シリンガー（Joseph Schillinger）は、オクターブで反復しないスケール構造を「第4グループ（Fourth Group）」と分類した。SUHMMLydianのような構造は、12音すべてを経過するまで（あるいはそれ以上）オクターブ内で完結しないため、この分類に当てはまる1。
    
3. 五度圏の三度ヴォイシング化
    
    前述の通り、数学的には五度堆積（Stack of Fifths）と同義であるため、この操作は「五度圏の構成音を三度堆積として再配置するアルゴリズム」とも定義できる。
    

### 2.4 類似した操作との比較

ユーザーは「類似した操作はあるか？」と問うている。この問いに対しては、**定音程積層（Constant Interval Stacking）**との対比が最も有益な洞察を与える。

- **全音音階（Whole Tone Scale） / オーギュメント・コード**
    
    - 操作：長三度（M3）のみを積み重ねる。
        
    - 結果：C - E - G# - C...
        
    - 特徴：3音で閉じる（閉ループ）。対称性が高く、調性が曖昧。コルトレーン・チェンジ（Coltrane Changes）の基礎。
        
- **ディミニッシュ・コード（Diminished 7th）**
    
    - 操作：短三度（m3）のみを積み重ねる。
        
    - 結果：C - Eb - Gb - A - C...
        
    - 特徴：4音で閉じる（閉ループ）。「オクタトニック・スケール」の骨格。
        
- **SUHMMLydian / SUHMMDorian（交互積層）**
    
    - 操作：M3とm3を交互に積む。
        
    - 結果：C - E - G - B - D...（完全五度の連鎖と同等）
        
    - 特徴：**閉じない（開ループ）**。12音すべてを網羅し、さらに微分音的な領域へ続く。
        

このように比較すると、交互積層操作の特異性は、その**非対称性（Asymmetry）**によって「閉じた円環」を回避し、調性音楽の基礎である「完全五度」を生成し続ける点にあることがわかる。

## 3. 拡張和音における変形理論：九の和音の声部連結と「偶奇」の対称性

レポートの後半では、ユーザーが提示した具体的なコード進行の分析に移る。ここでは、**九の和音（Ninth Chords / Pentads）**において、構成音の「奇数番目（Odd members）」と「偶数番目（Even members）」の対称的な保持・移動が見られる現象を、**ネオ・リーマン理論（Neo-Riemannian Theory, NRT）**およびその拡張理論を用いて厳密に定義する。

### 3.1 和音構成音の「偶奇」と機能的分類

分析に先立ち、ユーザーが指摘した「奇数番目」「偶数番目」という概念を音楽理論的に整理する。

- **奇数構成音（Odd Members）：Root (1), 5th (5), 9th (9)**
    
    - これらは伝統的に**構造音（Structural Tones）**と呼ばれることが多い。ルートと完全五度は和音の音響的土台を形成し、ナインス（9th）は五度の上にさらに五度を重ねた音（1 $\to$ 5 $\to$ 9）として、構造的な「柱」の延長にある。これらは倍音列においてより協和的な関係（完全音程）を持つ傾向がある。
        
- **偶数構成音（Even Members）：3rd (3), 7th (7)**
    
    - これらはジャズ理論において**ガイドトーン（Guide Tones）**と呼ばれる。和音の「質（Quality）」（メジャーかマイナーか、ドミナントかトニックか）を決定づける最も重要な音である。
        

ユーザーの問いは、この「構造音（Odd）」と「ガイドトーン（Even）」が互いに排他的に移動・保持される操作の名前を求めているものである。

### 3.2 ケースA：スライド変形（Slide Transformation / S操作）

**現象：**

- CM9 (C, E, G, B, D) $\to$ C#m9 (C#, E, G#, B, D#)
    
- **保持される音（偶数/Even）：** 3rd (E), 7th (B)
    
- **半音移動する音（奇数/Odd）：** Root (C$\to$C#), 5th (G$\to$G#), 9th (D$\to$D#)
    

#### 3.2.1 理論的定義：スライド変形（S）

この操作は、ネオ・リーマン理論における**[[SLIDE操作の定義|スライド変形]]（Slide Transformation、略称：S）**、あるいは**S操作**と呼ばれるものの拡張形である16。

本来の三和音（トライアド）におけるスライド変形は、**「第3音を共有する長三和音と短三和音の交換」**と定義される。

- **Triadic S:** C Major (C, **E**, G) $\longleftrightarrow$ C# Minor (C#, **E**, G#)
    
- ここで、ルートと五度は半音上に平行移動し、第3音（E）のみが共通音として保持される。
    

ユーザーの例（CM9 $\to$ C#m9）は、このS操作を**九の和音（Pentad）**へ厳密に拡張したものである。

- **第7音（7th）の保持：** C MajorにおけるB音（M7）と、C# MinorにおけるB音（m7）は、異名同音ではなく物理的に同一のピッチである。C#mにおけるBは「短七度」であるが、C#に対するB（短七度）とCに対するB（長七度）が結果的に同じ音高となるため、**第3音（3rd）と第7音（7th）の両方が共通音（Common Tones）として機能する**。
    
- **第9音（9th）の移動：** Cの9th（D）からC#の9th（D#）への移動は、ルートの移動（C$\to$C#）に追従する平行移動である。
    

#### 3.2.2 名称と分類

この進行を指す正式な名称は**スライド変形（Slide Transformation）**、あるいは拡張和音であることを明示するために**Slide$_9$**や**S-Transformation**と呼ばれる。

#### 3.2.3 ジャズ的文脈と音響心理

ジャズにおいては、この進行は**「共通ガイドトーン上の再和声化（Reharmonization over Static Guide Tones）」として解釈される。3rdと7th（EとB）という和声の骨格（シェル）を保持したまま、ルート・五度・九度という「構造体」を半音ズラす（Side-slipping）ことで、強烈な緊張感と浮遊感を生み出す。コリアーの理論やネオ・ソウルにおいては、この「共通音を軸にした急激な転調」が頻出する。

この操作が「上昇（C$\to$C#）」だけでなく「下降（Gm9$\to$GbM9）」でも成立するのは、S操作が対合（Involution）**、つまり「行って戻る」操作であり、方向性を持たない写像だからである（C MajorにSを適用すればC# Minorになり、C# MinorにSを適用すればC Majorに戻る）。

**回答1：** このコード進行操作は**スライド変形（Slide Transformation / S操作）**と呼ばれる。特に九の和音においては、ガイドトーン（3, 7）を保持し、構造音（1, 5, 9）を半音変位させる操作となる。

### 3.3 ケースB：平行変形（Parallel Transformation / P操作）

**現象：**

- CM9 (C, E, G, B, D) $\to$ Cm9 (C, Eb, G, Bb, D)
    
- **保持される音（奇数/Odd）：** Root (C), 5th (G), 9th (D)
    
- **半音移動する音（偶数/Even）：** 3rd (E$\to$Eb), 7th (B$\to$Bb)
    

#### 3.3.1 理論的定義：平行変形（P）

この操作は、ネオ・リーマン理論における最も基本的な操作の一つ、**平行変形（Parallel Transformation、略称：P）**、あるいは**P操作**である18。

三和音においては、P操作は「同主調（Parallel Key）への移行」、すなわち**「ルートと五度を共有する長三和音と短三和音の交換」**と定義される。

- **Triadic P:** C Major (C, E, G) $\longleftrightarrow$ C Minor (C, **Eb**, G)
    

ユーザーの例では、これが九の和音に適用されている。

- **第7音の移動：** 機能和声的な「同主調」の関係（CMaj7 $\to$ Cmin7）において、第7音もまた長七度（B）から短七度（Bb）へ変化する。これはP操作の自然な拡張である。
    
- **第9音の保持：** 長九度（D）は、メジャーコード（CM9）でもマイナーコード（Cm9）でも共通してダイアトニックなテンション（Avoid Noteにならない）であるため、そのまま保持される。
    

#### 3.3.2 名称と分類

この操作は**平行変形（Parallel Transformation）**、あるいは**同主調転換（Modal Mixture / Modal Interchange）**と呼ばれる。

#### 3.3.3 S操作との対称性（双対性）

ここで特筆すべきは、ユーザーが発見した**幾何学的な対称性（Symmetry）**である。

- **S操作（スライド）：** 偶数番目（3, 7）を固定し、奇数番目（1, 5, 9）を動かす。
    
- **P操作（平行）：** 奇数番目（1, 5, 9）を固定し、偶数番目（3, 7）を動かす。
    

この観察は極めて鋭い洞察である。ネオ・リーマン理論の拡張空間において、S操作とP操作は、和音の構成音を「構造（Odd）」と「質（Even）」に二分した際、互いに直交する（Orthogonal）、あるいは**双対的（Dual）**な関係にある操作であることがわかる。

P操作は和音の「場所（Root）」を変えずに「質（Quality）」を変える操作であり、S操作は和音の「質（Quality-defining 3rd）」を軸にして「場所（Root）」を劇的に変える操作である。

**回答2：** このコード進行操作は**平行変形（Parallel Transformation / P操作）**と呼ばれる。九の和音においては、構造音（1, 5, 9）を保持し、ガイドトーン（3, 7）を半音変位させる操作となる。

## 4. 類似した操作と理論的背景の統合

### 4.1 ネオ・リーマン理論のPLR群とその拡張

S操作やP操作に「類似した操作」として、ネオ・リーマン理論の基本操作群（PLR群）が存在する。これらはすべて、共通音を最大化する（Parsimonious Voice Leading）法則に基づいている。

1. **R操作（Relative / 平行調）：** C Major $\leftrightarrow$ A Minor
    
    - 共通音：C (1), E (3)
        
    - 移動：G (5) $\to$ A
        
2. **L操作（Leading-Tone / 導音交換）：** C Major $\leftrightarrow$ E Minor
    
    - 共通音：E (3), G (5)
        
    - 移動：C (1) $\to$ B
        

S操作（Slide）は、これらPLR群に次ぐ重要な操作であり、L、P、Rの組み合わせ（$L \cdot P \cdot R$）としても表現可能であるが、その「3度を共有しつつルートが半音ズレる」という特異な距離感から、独立した操作として扱われることが多い。

### 4.2 コリアーの「ネガティブ・ハーモニー」との関連

ユーザーが言及したS操作とP操作の対称性は、ジェイコブ・コリアーが提唱する**ネガティブ・ハーモニー**の軸理論とも整合する。

- P操作は、Root-5th軸（主音と属音）に対する鏡像反転的な性質を持つ（質の反転）。
    
- S操作は、3rd-7th軸（ガイドトーン）に対する鏡像反転的な性質を持つ（位置の反転）。
    

このように、提示されたコード進行は単なる偶発的なパターンではなく、12平均律という空間における数学的な対称操作の一形態として位置づけられる。

# 拡張三和音における変形操作とアルゴリズム的和声構築：SUHMMLydianからネオ・リーマン理論の拡張まで

## 1. 序論：現代和声のフロンティア

現代音楽理論の景観において、和声はもはや「トニック」や「ドミナント」といった機能的な重力のみによって支配されるものではない。20世紀後半から21世紀にかけて、ジャズの高度化、映画音楽における色彩的技法、そしてネオ・リーマン理論（Neo-Riemannian Theory）の台頭により、和音は「機能」としてではなく、幾何学的な空間内における「オブジェクト」として捉え直されつつある。

本報告書は、ユーザーにより提示された特定の和声現象――無限の三度堆積構造と、拡張和音（Extended Chords）における特定の声部連結（Voice Leading）――について、その名称、構造的定義、そして理論的背景を徹底的に究明するものである。

具体的には、ジェイコブ・コリアー（Jacob Collier）によってポピュラー化された「Super Ultra Hyper Mega Meta Lydian」という概念の数学的・アルゴリズム的本質を解明し、さらに九の和音（ナインスコード）における「奇数構成音」と「偶数構成音」の対称的な挙動を、ネオ・リーマン理論における「スライド（Slide）」および「平行（Parallel）」変形として体系化する。

本稿の目的は、単なる名称の特定にとどまらず、これらの操作がなぜ音楽的に成立するのか、その背後にある音響心理学的・数学的構造（五度圏との同型性やガイドトーンの保存則）を明らかにすることにある。

## 2. アルゴリズム的積層：無限三度連鎖の幾何学

ユーザーが最初に提示した「長三度と短三度を交互に積み重ねる」という操作は、伝統的な和音構築（3度堆積）の極致であり、現代的なアルゴリズム作曲や理論的遊戯の領域に属する。

### 2.1 SUHMMLydian：明るさの螺旋

#### 2.1.1 定義と名称

ルート（根音）から開始し、長三度（4半音）と短三度（3半音）を交互に積み重ねることで生成される和音構造は、Super Ultra Hyper Mega Meta Lydian（以下、SUHMMLydian）と呼ばれる1。

例としてCをルートとした場合、構成音は以下のようになる：

- C (Root)
    
- E (M3)
    
- G (P5) ← CからE(M3) + G(m3) = P5
    
- B (M7)
    
- D (M9)
    
- F# (#11)
    
- A (13)
    
- C# (#15)...
    

この名称は、ジェイコブ・コリアーが2010年代後半に自身のYouTubeチャンネルや講義で使用し始め、瞬く間にインターネット上の音楽理論コミュニティに広まったものである。名称に含まれる「Super Ultra...」という過剰な修飾は、このスケールが12音平均律の限界を超えて無限に拡張可能であることを示唆している。

#### 2.1.2 五度圏との同型性（Isomorphism）

この構造の最も重要な理論的側面は、**「長三度＋短三度 ＝ 完全五度」という等式にある。

$$M3 (4) + m3 (3) = P5 (7)$$

この単純な算術的事実は、交互三度積層が五度圏（Circle of Fifths）**と構造的に等価（同型）であることを意味する。SUHMMLydianの構成音（C, E, G, B, D, F#, A...）を五度圏上にプロットすると、Cから始まり時計回り（シャープ方向）に連続する音列（C-G-D-A-E-B-F#...）と完全に一致する。

通常の五度圏が「完全五度」という広い跳躍で円を描くのに対し、SUHMMLydianは「三度」という狭い歩幅でその間を埋めながら進む。その結果、この和音は五度圏のシャープ側（明るい側）の音をすべて内包することになり、「リディアン」の特性（#4）を極限まで強調した響きとなる6。

### 2.2 SUHMMDorian：暗さの螺旋

次に、ユーザーが問う「短三度と長三度を交互に積み重ねる」コード（例：C, Eb, G, Bb, D, F, A...）について解説する。

#### 2.2.1 定義と名称

この構造は、SUHMMLydianの積層順序を逆転させたものであり、**Super Ultra Hyper Mega Meta Dorian**（**SUHMMDorian**）と呼ばれる11。

- C (Root)
    
- Eb (m3)
    
- G (P5) ← CからEb(m3) + G(M3) = P5
    
- Bb (m7)
    
- D (M9)
    
- F (11)
    
- A (13)...
    

この構成音はドリアン・モード（C Dorian）と一致する。SUHMMLydianが五度圏の時計回り（シャープ方向）に進むのに対し、SUHMMDorianは**反時計回り（フラット方向）**に進む。

$$m3 (3) + M3 (4) = P5 (7)$$

積層の合計は同じ完全五度であるが、起点が短三度であるため、生成される和音の質はマイナー（短和音）となり、五度圏のフラット側（C-F-Bb-Eb...）の音を順次取り込んでいく。

#### 2.2.2 操作の一般名称

このような「特定のインターバル・パターンを反復して和音を生成する操作」には、いくつかの学術的・一般的な呼称が存在する。

1. Chain of Thirds（三度連鎖） / Infinite Chain of Alternating Thirds
    
    最も記述的な名称である。「Circle（圏）」ではなく「Chain（連鎖）」や「Spiral（螺旋）」が好まれるのは、このパターンがオクターブ内で閉ループを形成しない（無限に続く）ためである15。
    
2. Schillinger's Fourth Group（シリンガーの第4グループ）
    
    ジョセフ・シリンガーの体系において、ルートがオクターブで回帰しない音階構造は第4グループに分類される1。
    
3. Generative Algorithm via Fifths（五度による生成アルゴリズム）
    
    コリアー自身は、これを「五度圏を自然な形で拡張したもの」として捉えており、リディアンを「メジャースケールの自然な形（Natural form of Major）」と位置づけている7。
    

### 2.3 類似操作との比較

この「交互積層」の特異性を浮き彫りにするために、類似する「定音程積層」と比較する。

|**積層パターン**|**結果（C起点）**|**構造的特徴**|**名称**|
|---|---|---|---|
|**長三度のみ (M3)**|C, E, G#|3音で閉じる（閉ループ）|**オーギュメント・コード** / 全音音階の一部|
|**短三度のみ (m3)**|C, Eb, Gb, A|4音で閉じる（閉ループ）|**ディミニッシュ・コード** / オクタトニックの一部|
|**長・短交互 (M3, m3)**|C, E, G, B, D...|**閉じない（無限螺旋）**|**SUHMMLydian**（五度圏同型）|
|**短・長交互 (m3, M3)**|C, Eb, G, Bb, D...|**閉じない（無限螺旋）**|**SUHMMDorian**（五度圏同型）|

このように、交互積層操作のみが「閉じた円環」を脱し、全12音（さらにはそれ以上）を網羅する広大な和声空間を生成する。

## 3. 拡張和音の変形理論：S操作とP操作の対称性

ユーザーの後半の問いは、九の和音（Pentads）における特定の声部連結に関するものである。ここで提示された「奇数番目の構成音（Odd members）」と「偶数番目の構成音（Even members）」の挙動は、**ネオ・リーマン理論**における変形操作（Transformations）として極めて明快に説明できる。

### 3.1 構成音の機能分類：「構造」と「色彩」

分析の前提として、和音構成音をユーザーの指摘通りに分類する。

- **奇数グループ（Odd Parity）：Root(1), 5th(5), 9th(9)**
    
    - これらは「五度堆積（Stack of Fifths）」の系列にある（1 $\to$ 5 $\to$ 9）。和音の物理的な安定性やルートの確立に寄与する**構造音（Structure）**である。
        
- **偶数グループ（Even Parity）：3rd(3), 7th(7)**
    
    - これらは**ガイドトーン（Guide Tones）**として知られ、和音のメジャー/マイナー、ドミナント/トニックといった「質（Quality）」を決定する**色彩音（Color）**である。
        

### 3.2 ケースA：スライド変形（Slide Transformation / S）

**問い：**

> Rootから数えて偶数番目の構成音（3度・7度）を保持し、奇数番目の構成音（1度・5度・9度）を半音移動させるコード進行（例：CM9 $\to$ C#m9）をなんという？

回答：

この操作は[[SLIDE操作の定義|スライド変形]]（Slide Transformation）、あるいは単にS操作と呼ばれる16。

#### 3.2.1 理論的メカニズム

本来のネオ・リーマン理論におけるS操作は、三和音間で「第3音を共有し、ルートと第5音が半音平行移動する関係（例：C Major $\leftrightarrow$ C# Minor）」を指す。

ユーザーの例（CM9 $\to$ C#m9）は、これを九の和音へ拡張したものである。

- **第3音（E）の保持：** 定義通り、S操作の軸となる。
    
- **第7音（B）の保持：** ここが重要である。C MajorにおけるB（長七度）と、C# MinorにおけるB（短七度）は、物理的に同じ音高である。S操作においてルートが半音上昇（C$\to$C#）した結果、保持されたB音は「長七度」から「短七度」へと、その機能（インターバル）を変化させるが、音高そのものは**不変**である。これにより、**3度と7度という「ガイドトーンのペア」が完全に保持される**という現象が起きる。
    
- **ルート・5度・9度の移動：** これら奇数グループは、すべて半音上（C$\to$C#, G$\to$G#, D$\to$D#）へ平行移動する。
    

#### 3.2.2 音楽的意義

この進行は「ガイドトーンを共有する遠隔調への転調」として、現代ジャズやネオ・ソウルで多用される。聴覚上は、和声の骨格（3, 7）が維持されているため滑らかに聞こえるが、ルートが半音ズレるため「足元が滑る（Slide）」ような浮遊感をもたらす。

### 3.3 ケースB：平行変形（Parallel Transformation / P）

**問い：**

> Rootから数えて奇数番目の構成音（1度・5度・9度）を保持し、偶数番目の構成音（3度・7度）を半音移動させるコード進行（例：CM9 $\to$ Cm9）をなんという？

回答：

この操作は平行変形（Parallel Transformation）、あるいはP操作と呼ばれる18。また、機能和声の文脈では**同主調借用（Modal Interchange / Mode Mixture）**とも呼ばれる。

#### 3.3.1 理論的メカニズム

P操作は「同じルートを持つメジャーコードとマイナーコードの変換」である。

- **ルート・5度・9度の保持：** これら奇数グループは、メジャー（CM9）でもマイナー（Cm9）でも共通してダイアトニックな構成音であるため、保持される。特に第9度（D）がマイナーコードにおいても長九度（Major 9th）として保持される点は、ジャズ・ハーモニーにおけるCm9の特徴である。
    
- **3度・7度の移動：** メジャー（E, B）からマイナー（Eb, Bb）へ、偶数グループが一斉に半音下降（あるいは上昇）する。これにより和音の「質」が反転する。
    

#### 3.3.2 S操作との対称性

ユーザーの指摘する通り、S操作とP操作は、九の和音空間において見事な**双対性（Duality）**を示している。

|**変形操作**|**保持される音群**|**移動する音群**|**変化する属性**|
|---|---|---|---|
|**P操作（平行）**|奇数群（構造：1, 5, 9）|偶数群（色彩：3, 7）|和音の「質」（Maj $\leftrightarrow$ Min）|
|**S操作（スライド）**|偶数群（色彩：3, 7）|奇数群（構造：1, 5, 9）|和音の「位置」（Root C $\leftrightarrow$ C#）|

この対称性は、和声空間における「構造」と「色彩」が独立して操作可能なパラメータであることを示唆しており、コリアーのネガティブ・ハーモニー理論（軸による反転）とも深く共鳴するものである。

## 4. 結論

以上の分析より、ユーザーの問いに対する最終的な回答は以下の通りである。

1. **交互三度積層コード（M3+m3）：** **SUHMMLydian（スーパー・ウルトラ・ハイパー・メガ・メタ・リディアン）**。操作名は**Chain of Alternating Thirds（交互三度連鎖）**あるいは**五度圏のアルゴリズム的生成**。
    
2. **逆交互三度積層コード（m3+M3）：** **SUHMMDorian（スーパー・ウルトラ・ハイパー・メガ・メタ・ドリアン）**。
    
3. **進行1（3, 7保持）：** **[[SLIDE操作の定義|スライド変形]]（Slide Transformation / S操作）**。
    
4. **進行2（1, 5, 9保持）：** **平行変形（Parallel Transformation / P操作）**。
    

これらの概念は、音楽を静的な「響き」としてではなく、動的な「操作」と「変換」のプロセスとして捉える現代音楽理論の精髄を表している。
